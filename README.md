# twitter-clone-project
This is a python(django) project which aims to clone some of the functionalities of twitter 

1. Users
   -> Register
   -> Login
   -> Logout
   -> Profile
     -> Image
     -> Text
     -> Follow Button
    -> Feed
     -> User's feed only
     -> User's + who they follow
2. Tweets
   -> User Permissions
   -> Creating
     -> Text
     -> Image
   -> Delete tweet
   -> Retweeting 
   -> Liking & Unliking a tweet
3. Following / Followers 

GOING FOWARD, MORE FUNCTIONS TO BE ADDED
1. Messaging between users
2. Push Notifications on the frontend 
3. Explore (finding hash tags)

5. Video call
